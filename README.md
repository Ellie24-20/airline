# 航空旅行一站式服务平台

## 项目简介
基于Spring Cloud微服务架构和Vue.js前端的综合性旅行预订平台

## 技术栈
- 后端：Spring Boot, Spring Cloud, MySQL, Redis
- 前端：Vue.js, TypeScript, Element Plus
- 部署：Docker, Kubernetes
